# Hangman Game

## Overview
A console-based word guessing game developed using Python. Players guess letters to reveal a hidden word within a limited number of attempts.

## Features
- Random word selection
- Letter-by-letter guessing
- Limited attempts
- Win/Loss conditions

## Technologies Used
- Python
- VS Code

## Learning Outcomes
- Loops and Conditions
- String Manipulation
- User Input Handling
